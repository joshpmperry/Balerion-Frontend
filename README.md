# Balerion-Frontend


### Using MERN

- MongoDB
- Express.js
- React
- Node.js


Installation 

> Install node modules

```bash
npm install
```

Run Backend

> Runs using nodemon for hot reload

```bash
npm run dev
```

Run Frontend

```bash
cd ./frontend

npm run dev
```
